# 📝 TaskManager: Gestor de Tareas con Flask

Una aplicación web robusta y minimalista para la gestión de tareas diarias, diseñada para ser rápida, intuitiva y fácil de desplegar.



[Image of Task Management Web App Interface]


## 🚀 Características
* **CRUD Completo:** Crea, visualiza, edita y elimina tareas sin complicaciones.
* **Persistencia de Datos:** Uso de base de datos relacional para mantener tus tareas a salvo.
* **Diseño Responsivo:** Interfaz moderna construida con Tailwind CSS, adaptable a móviles y escritorio.
* **Backend Eficiente:** Lógica de servidor optimizada con Flask.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python
* **Framework Web:** Flask
* **ORM:** SQLAlchemy (Base de datos)
* **Estilos:** Tailwind CSS
* **Plantillas:** Jinja2 (HTML dinámico)
